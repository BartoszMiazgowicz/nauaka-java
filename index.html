2 DataWaznosci.java

Klasa package-private (bez public) — spełnia wymóg 6.3: niewidoczna spoza pakietu.

📄 DataWaznosci.java
// 6.3: brak public → package-private (niewidoczna spoza pakietu)
class DataWaznosci {
  private int dzien, miesiac, rok;

  DataWaznosci(int dzien, int miesiac, int rok) {
    System.out.println("[DataWaznosci] konstruktor: " + dzien + "/" + miesiac + "/" + rok);
    this.dzien = dzien;  this.miesiac = miesiac;  this.rok = rok;
  }

  // konstruktor kopiujący — głęboka kopia dla kompozycji
  DataWaznosci(DataWaznosci other) {
    System.out.println("[DataWaznosci] konstruktor kopiujący");
    this.dzien = other.dzien;  this.miesiac = other.miesiac;  this.rok = other.rok;
  }

  @Override
  public String toString() { return dzien + "/" + miesiac + "/" + rok; }
}

🛒 CZĘŚĆ 3 — Towar.java (zadania 6.0, 6.1, 6.2, 6.3)

3 Towar.java

📄 Towar.java
import java.util.*;

// 6.3: package-private (bez public) — Towar niewidoczny spoza pakietu
class Towar {

  // ── 6.1: pola statyczne i niestatyczne ──────────────────────────────
  // ✓ static: jedno pole dla WSZYSTKICH obiektów → liczy sumę stworzonych
  private static int countStatic_ = 0;
  // ✗ countNonStatic_: każdy obiekt ma własną kopię (zawsze = 1) — wykomentowane
  // public int countNonStatic_ = 0;

  // ✓ instance: własna kopia w każdym obiekcie → unikalny ID zachowany
  private int idNonStatic_;
  // ✗ idStatic_: wszyscy widzą tę samą wartość (ostatnią) — wykomentowane
  // public static int idStatic_ = 0;

  // ── 6.3: private — dostęp tylko wewnątrz klasy Towar ────────────────
  private String nazwa;
  private double cena;
  private int ilosc;
  private DataWaznosci dataWaznosci; // 6.2: kompozycja

  // 6.3: getter — Sklep może ODCZYTAĆ countStatic_, nie może go zmienić
  static int getCount() { return countStatic_; }

  // ── 6.0: konstruktor domyślny ────────────────────────────────────────
  Towar() {
    System.out.println("[Towar] konstruktor domyślny");
    this.nazwa = "ERROR";  this.cena = -1;  this.ilosc = -1;
    this.dataWaznosci = new DataWaznosci(1, 1, 1970);
    countStatic_++;  this.idNonStatic_ = countStatic_;
  }

  // ── 6.0: konstruktor z parametrami ──────────────────────────────────
  Towar(String nazwa, double cena, int ilosc, DataWaznosci data) {
    System.out.println("[Towar] konstruktor z parametrami: " + nazwa);
    this.nazwa = nazwa;  this.cena = cena;  this.ilosc = ilosc;
    this.dataWaznosci = data;
    countStatic_++;  this.idNonStatic_ = countStatic_;
  }

  // ── 6.0: konstruktor kopiujący ───────────────────────────────────────
  Towar(Towar other) {
    System.out.println("[Towar] konstruktor kopiujący: " + other.nazwa);
    this.nazwa = other.nazwa;  this.cena = other.cena;  this.ilosc = other.ilosc;
    this.dataWaznosci = new DataWaznosci(other.dataWaznosci); // głęboka kopia!
    countStatic_++;  this.idNonStatic_ = countStatic_;
  }

  // ── 6.0: wyświetl wszystkie pola ─────────────────────────────────────
  void wyswietl() {
    System.out.println("=== Towar [ID=" + idNonStatic_ + "] ===");
    System.out.println(" Nazwa: " + nazwa);
    System.out.println(" Cena: " + cena + " zł");
    System.out.println(" Ilość: " + ilosc + " szt.");
    System.out.println(" Data ważności: " + dataWaznosci);
  }
}

🏪 CZĘŚĆ 4 — Sklep.java + wyjaśnienia (6.0 użycie, 6.1)

Klasa główna z main — obiekty osobno, tablica, ArrayList i pole statyczne.

4 Sklep.java

📄 Sklep.java
import java.util.*;

public class Sklep {
  public static void main(String[] args) {

    // ── 6.0: obiekty stworzone "osobno" ─────────────────────────────
    Towar itemBlad = new Towar();  // konstruktor domyślny
    Towar itemPomidor = new Towar("Pomidor", 2.99, 100,
        new DataWaznosci(20, 4, 2026)); // z parametrami
    Towar itemPomidorGalazka = new Towar(itemPomidor); // kopiujący

    itemBlad.wyswietl();
    itemPomidor.wyswietl();
    itemPomidorGalazka.wyswietl();

    // ── 6.0: tablica ────────────────────────────────────────────────
    Towar[] tab = new Towar[3];
    // Po new Towar[3]: tab[0] == null
    // System.out.println(tab[0]) → "null" (nie crashuje)
    // tab[0].wyswietl() → NullPointerException!
    Arrays.fill(tab, new Towar("Marchew", 1.50, 50,
        new DataWaznosci(15, 5, 2026))); // ⚠ 3× ta sama referencja!
    for (Towar t : tab) t.wyswietl();

    // ── 6.0: kolekcja ───────────────────────────────────────────────
    ArrayList<Towar> list = new ArrayList<Towar>();
    list.add(new Towar());  // domyślny
    list.add(new Towar("Jabłko", 3.20, 200,
        new DataWaznosci(10, 6, 2026))); // z parametrami
    list.add(new Towar(itemPomidor)); // kopiujący
    for (Towar t : list) t.wyswietl();

    // ── 6.1: countStatic_ BEZ użycia obiektu ────────────────────────
    System.out.println("Łącznie stworzono: " + Towar.getCount() + " towarów");
  }
}

5 Wyjaśnienie — static vs niestatyczne (6.1 + 6.3)

✓ countStatic_  static  → jedno pole dla WSZYSTKICH obiektów → poprawnie liczy sumę
✗ countNonStatic_ instance → każdy obiekt ma WŁASNĄ kopię (zawsze = 1) → nie działa

✓ idNonStatic_  instance → własna kopia = przypisany ID zachowany w obiekcie → działa
✗ idStatic_  static  → wszyscy widzą tę samą wartość (ostatnią) → nie daje unikalnych ID
